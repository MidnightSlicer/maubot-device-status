# maubot-webhook
A [maubot](https://github.com/maubot/maubot) to send messages using webhooks.

## Building
Use the `mbc` tool to build this plugin:
```
mbc build
```
Optionally use the `-u` switch to upload it to your maubot instance, if configured:
```
mbc build -u
```
